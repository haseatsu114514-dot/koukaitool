# 生年月日 逆算くん

「年齢」と「誕生日」だけ分かる相手の生年月日を逆算できる公開向けシンプルツールです。  
相手の基本情報から素早く確認する用途を想定しています。

## コンセプト
- 誰でもURLを知っていれば使える（ログイン不要）
- 入力は `年齢` と `誕生日（月日）` のみ
- 余計な機能を省いた軽量UI

## 機能
- 生年月日の逆算（2人まで）
  - 1人目は必須、2人目は「+ 2人目を追加」で任意入力
- 1人目の年齢・誕生日が有効入力になると自動で逆算
- 入力中のリアルタイムチェック
  - 年齢（0〜130）
  - 誕生日（月/日）の形式
- うるう日（2/29）対策
  - 2/29入力時は平年の年齢判定差を自動考慮し、必要に応じて候補を表示
- 結果コピー
  - 1人目 / 2人目の個別コピー
- 各ボックスで個別クリア
- ページ開きっぱなしで日付が変わっても、基準日を自動更新
